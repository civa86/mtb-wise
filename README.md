# MTB Wise

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```

## User Data

```bash
# Mac OSX
open ~/Library/Application\ Support/mtb-wise
```

## App Icon

Design icon with 1024x1024 resolution and save it: `resources/icon.png`

Run `./icon-make.sh` script

The script requires [ImageMagik](https://imagemagick.org/)
