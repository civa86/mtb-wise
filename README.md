# MTB Wise

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```

## User Data

```bash
# Mac OSX
open ~/Library/Application\ Support/mtb-wise
```
